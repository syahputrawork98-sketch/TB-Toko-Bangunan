# Manajemen Proyek & Alur Kerja (Project & Workflow Management)

Folder ini berfungsi sebagai pusat kendali manajemen proyek dan aturan kerja pengembang (AI & Manusia) dalam sistem **TB-Toko-Bangunan**.

---

## 📁 Struktur Folder
*   **`onboarding/`**: Berisi berkas panduan inisiasi sesi chat AI baru:
    *   `CHATGPT_PROJECT_INSTRUCTIONS.md`: Custom instructions untuk ChatGPT.
    *   `ROOM_00_MANAGER_PROMPT.md`: Prompt inisial untuk mengaktifkan peran Manager.
    *   `ROOM_01_REVIEWER_PROMPT.md`: Prompt inisial untuk mengaktifkan peran Reviewer.
*   **`workflow/`**: Berisi panduan pelaksanaan tugas harian:
    *   `WORKING_SYSTEM.md`: Prinsip utama alur kerja, ukuran batch, kriteria selesai (*Definition of Done*), dan kebijakan model AI.
    *   `EXISTING_MODE.md`: Aturan audit penemuan fitur (*Discovery-First*) sebelum mengubah kode.
    *   `PROJECT_STARTER_CHECKLIST.md`: Daftar periksa sebelum memulai pengerjaan batch baru.

---
*Catatan: Pola sistem ini diadaptasi dari referensi template manajemen WK-Workflow-Kit.*
